# Liquid Glass UI

Beautiful, accessible Liquid Glass components for React with real SVG/CSS
refraction and production-ready fallbacks.

Use this Storybook as the visual documentation site for
`@clean99/liquid-glass`. The repository README stays short, the Markdown docs
carry governance and release details, and this site gives maintainers and users
a component-first entry point.

## Visual Documentation Contract

Every public component should have evidence for light and dark themes, reduced
motion, high contrast, mobile layout, and fallback or solid mode where relevant.
Kube-aligned primitives also need reference stories and strict comparison gates.

The source of truth is `docs/visual-state-coverage.json`, validated by
`pnpm test:visual-docs`.

## Release Gate

```sh
pnpm format
pnpm lint
pnpm typecheck
pnpm test:docs
pnpm test:release-readiness
pnpm test:unit
pnpm verify
```

Do not claim npm publish, GitHub Pages deployment, or exact Kube 1:1 parity
until the matching command or remote run has succeeded.
